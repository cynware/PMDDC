## A small program where you can create your own Pokémon Mystery Dungeon dialogue!

> Created by Rem & Cyn

Welcome to the world of Pokémon Mystery Dungeon: Dialog Creator!
This program is made to allow people to create their own dialogue in the style of Pokémon Mystery Dungeon: Explorers of Sky,
and has been designed to be as accurate as possible, down to the **pixel**.

## FEATURES!
- User-friendly interface!
- Loading portraits of every single Pokémon from the [PMDCollab](https://sprites.pmdcollab.org/) repository!
- Full compatibility for custom portraits and dialogue borders!
- Pixel Perfect exports!
- Quickly swap characters with presets!
